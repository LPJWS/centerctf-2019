Burt
=====================
>Keep the line generator and the input file, but something is wrong with the generator ... Repair generator to find out the hidden message ...

Даётся простенькая программа на языке си, которая получает на вход текстовый файл и выдаёт нам строку из 19 символов... Эта строка генерируется из рандомных символов из
текстового файла... Рандом в программе представлен с помощью функции rand(), которая
зависит от значения из функции srand(), изначально в программе стоит одно значение,
не важно какое. Для того, чтобы получить флаг надо перебрать примерно 10000 значений,
для функции srand()... Далее мы записываем все значения в текстовый файл и грепаем,
по слову flag. Получается, что только при одном значении функции srand() получаем
валидную строку.
