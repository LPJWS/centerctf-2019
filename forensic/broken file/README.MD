broken file
=====================
>In general, we rummaged around on hard disks and found a file, only the problem is that it does not open. Try it, maybe it will work out for you.

Суть данного задания в том чтобы попробовать восстановить его заголовок и расширение, данное задание решается просто подбором. Когда восстановишь файл и получишь jpg нужно будет посмотреть вдаль там будет написан флаг.
